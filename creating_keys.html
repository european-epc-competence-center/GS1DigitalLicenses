Creating Keys

This section describes how to create Digital GS1 Key Credentials.

Key Overview

Section described the various GS1 ID Key types that can be created from a GCP.

A MC that has obtained a GCP or ID Key License can create GS1 key Credentials to assert the existence of a specific GS1 ID Key. For example, if a fictictious company Healthy Tots has a Digital GCP license and wishes to commision a GTIN from their GCP, they can create and share a GS1 Key Credential asserting the existence of this GTIN for global trade.

If an MC uses serialization, batch/lot or other identifiers to further specify their individual trade items, the MC can also generate specific GS1 Key Credentials to assert the existence of a specific batch/lot or serial number of a trade item.

The specific format of each GS1 ID Key is different. this quick reference card higlights the basics of how to fornat GS1 ID Keys from a GCP. For more details on how to form individual GS1 ID Keys from your GCP, see your MO web page, the General Specification or services like GS1 Activate

A GS1 ID Key License Credential still requires a GS1 Key Credential to assert that the GS1 ID Key is in circulation. Think of a GS1 ID Key License Credential as a right to use a specific GS1 ID Key and the GS1 Key Credential as the actual instantiation (usage) of that Key.

Key Credential Overview

GS1 Key Credentials use the W3C Verifiable Credentials Data Model Version 2.0. The VCDM credential format uses JSON-LD to encapsulate the details of a license in the format that is well understood worldwide. A few optional properties of the W3C VCDM are mandatory in GS1 Key Credentials. These are outlined here:

Key Credential ID Property

The Key Credential ID Property SHOULD be a URI selected to be globally unique. If its desired to have the credential publicly accessible the credential ID Property SHOULD be a resolvable URL.

Key CredentialSubject ID Property

The Key Credential subject ID Property MUST be present and MUST be a GS1 Digital Link URI.

All GS1 Key Credentials share these properies:
